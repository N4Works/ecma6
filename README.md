# Configuracao necessaria

- Node 5
- Babeljs 5.8
- Typescript 1.6

# Instalacao

### Download do Node:

https://nodejs.org/en/


### Download do Babel:

```shell
$ npm i -g babel@5
```

### Download do Typescript:

```shell
$ npm i -g typescript

$ npm i -g ts-node
```


# Rodando os exemplos

### Node:

```shell
$ node node/INSIRA_O_NOME_DO_ARQUIVO_AQUI.js
```


### Babel:

```shell
$ babel-node babel/INSIRA_O_NOME_DO_ARQUIVO_AQUI.js
```


### Typescript:

```shell
$ ts-node typescript/INSIRA_O_NOME_DO_ARQUIVO_AQUI.ts
```
